void LockDown::removeDoor ( Door *  door)

Removes a door from the lockdown list.

This method removes the specified door from the doors collection if it is not a null pointer. It iterates through the list of doors, searching for a matching door, and erases it from the collection once found.

Parameters
doorA pointer to the Door object to be removed from the list. If the pointer is null, the method does nothing.
