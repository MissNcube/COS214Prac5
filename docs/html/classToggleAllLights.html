void ToggleAllLights::execute ( )
virtual

Toggles the state of all lights.

This method iterates through all the lights managed by the ToggleAllLights object and calls the toggle() method on each one. It also outputs messages to indicate when the toggling starts and ends.

Note
If a light is currently on, it will be turned off and vice versa.
See also
Light::toggle() for the method that changes the state of a light.
ToggleAllLights::addLights() for adding lights to the toggle list.
ToggleAllLights::removeLights() for removing lights from the toggle list.
