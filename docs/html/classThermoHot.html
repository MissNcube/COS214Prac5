void ThermoHot::heat ( Thermostat *  thermostat)
virtual

Increases the temperature of the thermostat when it is already hot.

This method is called when an attempt is made to heat the thermostat that is already in the "Hot" state. It outputs a message indicating that the thermostat is already hot and increases its temperature by 10 degrees.

Parameters
thermostatPointer to the Thermostat object that is being acted upon.
See also
ThermoHot::cool() for cooling the thermostat.
ThermoHot::idle() for changing the state to idle.
