ThermoIntegrator::ThermoIntegrator ( LegacyThermostat *  legacy)

Constructs a ThermoIntegrator instance.

This constructor initializes the ThermoIntegrator with a given legacy thermostat and sets the initial state to ThermoIdle.

Parameters
legacyA pointer to a 'LegacyThermostat' object that this integrator will manage.
Note
The initial state is set to ThermoIdle, which indicates that the system is not currently active.

◆ update()

void ThermoIntegrator::update ( )
virtual

Reacts to changes received from a connected sensor.

This method is called when the sensor notifies the thermostat of an update. The thermostat checks the current temperature and adjusts its state accordingly:

  • If the temperature is below 15°C, it switches to heating mode.
    See also
    ThermostatState::heat()
  • If the temperature is above 32.5°C, it switches to cooling mode.
    See also
    ThermostatState::cool()
  • If the temperature is within a comfortable range, it idles.
    See also
    ThermostatState::idle()
    The current state and temperature are then displayed.
