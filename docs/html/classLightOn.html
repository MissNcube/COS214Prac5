void LightOn::off ( Light *  light)
virtual

Turns the light off and changes its state.

This method handles the action of turning off the light. It outputs a message indicating that the light has been turned off and changes the light's state to LightOff.

Parameters
lightPointer to the Light object that is being acted upon.
See also
LightOn::on() for turning on the light.

void LightOn::on ( Light *  light)
virtual

Handles the action when the light is already on.

This method is called when an attempt is made to turn on a light that is already in the "On" state. It outputs a message indicating that the light is already on and does not change the state.

Parameters
lightPointer to the Light object that is being acted upon.
See also
LightOn::off() for turning off the light.
