void UnlockDoors::execute ( )
virtual

Unlocks all doors.

This method iterates through all the doors managed by the UnlockDoors object and calls the unlock() method on each door that is not null. It also outputs messages to indicate when the unlocking starts and ends.

Note
Only non-null doors will be unlocked.
See also
Door::unlock() for the method that changes the state of a door.
UnlockDoors::addDoor() for adding doors to the unlock list.
UnlockDoors::removeDoor() for removing doors from the unlock list.
